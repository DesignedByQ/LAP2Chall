### Introduction

The is an app used for posting stories.

### Tech Used

1. Node.js
2. Express
3. Nodemon
4. JavaScript
5. HTML
6. CSS
7. Mongo

### Installation

Navigate to the 'api' folder and run 'npm install' in the terminal. Run 'npm start' in terminal to get the server going, then open up http://localhost:3000 in the browser.
